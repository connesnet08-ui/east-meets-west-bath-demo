# East Meets West Bath — concept preview

Mobile-first one-page concept preview for East Meets West, 33 Southgate Street, Bath.

- Not the restaurant's current official website.
- Search engines should not index it.
- Uses lightly refined real imagery from the restaurant's current official website; sources and refinements are recorded in `image-sources.md`.
- Does not repeat prices from the restaurant's 2022-metadata menu PDFs.
- Does not claim instant booking, live availability, ratings, awards or customer reviews.
- Static deployment: publish this directory with no build command.
